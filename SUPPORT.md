# Support

OpenSyria Datasets API is public and maintainer-led.

## Where to Ask

Use GitHub issues in this repository for:

- reproducible API bugs,
- stale or unclear API documentation,
- deployment, Docker, CI, Redis, database, dataset-sync, or read-model import problems,
- maintainer proposals for API or tooling changes.

Use the relevant dataset repository for:

- incorrect records,
- missing records,
- source corrections,
- translation or alias corrections,
- dataset schema proposals.

For geography data, use the `data-geography` repository.

## Security

Do not report security vulnerabilities in public issues. Follow [SECURITY.md](SECURITY.md).

## Response Expectations

This project is maintained by a small maintainer team. Issues may be triaged by priority, project scope, reproducibility, and safety impact.

Maintainers may close issues that are out of scope, duplicates, missing reproduction details, or better handled in a dataset repository.
